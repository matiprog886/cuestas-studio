Preguntas frecuentes.

¿Por qué desde USD 289?

Es el punto de partida para una landing a medida bien hecha: diseño propio, optimización móvil, SEO de base y conexión con WhatsApp. A partir de ahí, el precio crece según el alcance que tu proyecto necesite.

¿El precio es cerrado o por hora?

Cerrado. Antes de empezar te paso un presupuesto fijo por el proyecto completo, con todo lo que incluye. Sin sorpresas ni horas extra inesperadas.

¿Incluye el dominio y el hosting?

El sitio se entrega listo para publicar. El dominio y el hosting se contratan aparte (son de bajo costo) y te acompaño en todo el proceso para dejarlo funcionando.

¿Hay costos mensuales?

No es obligatorio. El sitio es tuyo. Si querés, ofrezco mantenimiento y soporte continuo aparte, pero no es una atadura para tener tu web online.

¿Cuántas revisiones incluye?

Las necesarias para dejar el diseño como lo querés. Trabajamos sobre una propuesta y la ajustamos juntos antes de pasar al desarrollo.
