# Cuestas Studio — Landing

Sitio estático (HTML/CSS/JS, sin build) de la landing de Cuestas Studio.

- `index.html` — página principal
- `css/` — `styles.css` (sistema de diseño y tokens) + `sections.css` (secciones y componentes)
- `js/app.js` — interacciones (scroll suave Lenis, reveals, transiciones de bloque, menú mobile, contadores)
- `assets/` — logo CS (transparente), retrato y favicons

Desplegado en Vercel con auto-deploy desde este repositorio (push a `main` publica a producción).

URL de producción: https://cuestas-studio.vercel.app
